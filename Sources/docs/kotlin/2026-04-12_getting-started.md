## Get started with Kotlin

Kotlin is a modern language that's concise, multiplatform, and interoperable with Java and other languages.

## Install Kotlin

Kotlin is included in each IntelliJ IDEA and Android Studio release.

## Use cases

### Backend (Server-side)
- Configure a Java project to work with Kotlin
- Create a RESTful web service with Spring Boot
- Create HTTP APIs with Ktor

### Multiplatform
- Share business logic while keeping the UI native
- Share business logic and UI with Compose Multiplatform
- Make existing Android application work on iOS

### Android
- Google recommends Kotlin as the primary language for Android development

### Data Science
- Kotlin Notebook for data exploration
- DataFrame library for data analysis
- Kandy for data visualization
