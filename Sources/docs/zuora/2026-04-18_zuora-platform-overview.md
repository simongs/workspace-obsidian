# Zuora Platform overview — 공식 문서 원문

> 2026-04-18 수집. Last updated: March 23, 2026

## 공식 본문

> Zuora Platform is the foundation for the Zuora product suite, letting IT and Engineering teams easily connect, scale, and deploy alongside the wider monetization systems architecture.

## 7개 capability 카테고리 (공식 테이블)

| 영역 | 하위 기능 |
|------|---------|
| **Getting Started — System Management** | Command Center · Settings Service · System Health |
| **Getting Started — Data Management** | Data Loader · Deployment Manager · DevOps · Environments |
| **Getting Started — Organization & Entity Management** | Multi-Org · Multi-Entity |
| **Extensibility** | Attachments · Custom Fields · Custom Logic · Custom Objects · Events and Notifications · Mediation · Object Manager · Workflow |
| **Integration** | APIs · Connector for Data Warehouses · Integration Hub · SDKs · Secure Data Share for Snowflake |
| **Data** | Analytics · Data Query · Data Warehouse · In-product Benchmark · Object Manager · Reporting |
| **Security and Identity** | Administrator Settings · Audit Trail · Zuora OneID · Data Protection and Privacy · Event Streaming Connector · Zuora Protect |
