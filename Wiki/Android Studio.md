## 개요

Android Studio는 Google이 배포하는 Android 앱 공식 개발 IDE다. [[JetBrains]]의 [[IntelliJ IDEA]] Community 에디션 위에 Android 전용 도구(에뮬레이터, 레이아웃 편집기, APK 분석 등)를 얹은 구조다.

## Kotlin과의 관계

- [[Kotlin]]이 **기본 내장** — 새 프로젝트의 기본 언어
- Google이 2019년부터 Android 개발의 **공식 권장 언어**로 Kotlin을 지정
- Jetpack Compose(선언적 UI 툴킷)가 Kotlin 기반

## Android 전용 기능

| 기능 | 용도 |
|------|------|
| AVD Manager | 가상 디바이스 에뮬레이터 관리 |
| Layout Inspector | 런타임 UI 계층 분석 |
| Profiler | CPU/메모리/네트워크 프로파일링 |
| APK Analyzer | 빌드 산출물 크기/구성 분석 |
