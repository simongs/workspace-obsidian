## 개요

IntelliJ IDEA는 [[JetBrains]]가 개발한 JVM 계열 언어 전용 IDE다. Java, [[Kotlin]], Scala, Groovy 등을 지원하며, JetBrains의 대표 제품이다.

## Kotlin 지원

- Kotlin 컴파일러와 언어 서버가 **기본 탑재** — 별도 플러그인 불필요
- 코드 완성, 리팩토링, 디버거가 Kotlin에 완전 대응
- Java ↔ Kotlin **자동 변환** 기능 내장 (클립보드에 Java 코드를 복사 후 Kotlin 파일에 붙여넣으면 변환)
