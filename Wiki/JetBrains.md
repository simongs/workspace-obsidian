## 개요

JetBrains는 체코 프라하에 본사를 둔 소프트웨어 회사로, 개발자용 IDE와 [[Kotlin]] 프로그래밍 언어로 잘 알려져 있다. 2000년 Sergey Dmitriev, Valentin Kipiatkov, Eugene Belyaev가 설립했다.

## Kotlin과의 관계

- 2010년 [[Kotlin]] 개발 착수, 2016년 1.0 릴리즈
- 목표: 자사의 Java 기반 IDE 개발 생산성 향상
- 이후 오픈소스화 (Apache 2.0), Kotlin Foundation을 Google과 공동 설립
- Kotlin 컴파일러, 표준 라이브러리, 공식 문서 유지보수

## Kotlin 생태계 확장

JetBrains는 Kotlin을 언어에 그치지 않고 프레임워크 레이어까지 직접 제공:

- **Ktor** — 백엔드 웹 프레임워크
- **Compose Multiplatform** — 선언적 UI 툴킷 (iOS/Android/데스크톱 공유)
- **Kotlin Multiplatform (KMP)** — 비즈니스 로직 크로스플랫폼 공유
- **Kotlin Notebook, DataFrame, Kandy** — 데이터 사이언스 도구
